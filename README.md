# STRANGER-THINGS-PERFORMANCE-DASHBOARD
📌 About the Project

The Stranger Things Analytics Dashboard is an interactive Power BI project that analyzes the performance of the Netflix series Stranger Things across all seasons and episodes. The dashboard provides insights into IMDb ratings, audience voting trends, highest-rated episodes, and fan-favorite characters using visually engaging and theme-based analytics.

Designed with a cinematic Stranger Things-inspired red and black aesthetic, the dashboard combines storytelling and data visualization to create an immersive user experience. Users can explore season-wise performance, compare episode ratings, identify the most popular characters, and discover key trends through interactive filters and dynamic visuals.

This project demonstrates skills in Power BI, DAX, data modeling, data visualization, dashboard design, and storytelling with data, while showcasing how entertainment data can be transformed into meaningful insights through business intelligence tools.
